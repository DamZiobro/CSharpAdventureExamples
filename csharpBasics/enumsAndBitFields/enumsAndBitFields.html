Enumeration is basic C# type mostly used for increase readability of code and make variable type safety.
Assume that we would like to use weekdays in some variable. We can create create int variable for it and assume that following numers (1-7) are associated with following weekdays. This would be enough but every use which will read our code should know about our assumptions to have no problems with analyze. Additionally someout can accidentialy assign value being out of range 1-7 which can cause unexpected problems.
